# Mandu Deployment - Rule Categories

## Build
프로덕션 빌드 최적화 및 번들링 설정

| Impact | Description |
|--------|-------------|
| CRITICAL | 빌드 실패 시 배포 불가 |
| HIGH | 번들 크기, 빌드 시간에 큰 영향 |

## Platform
배포 플랫폼별 설정 및 최적화

| Impact | Description |
|--------|-------------|
| CRITICAL | 플랫폼 호환성 필수 |
| HIGH | 성능, 비용에 큰 영향 |

## Container
Docker 컨테이너화 및 오케스트레이션

| Impact | Description |
|--------|-------------|
| HIGH | 일관된 배포 환경 보장 |
| MEDIUM | 개발/프로덕션 환경 일치 |

## CI/CD
지속적 통합 및 배포 파이프라인

| Impact | Description |
|--------|-------------|
| HIGH | 자동화된 품질 보증 |
| MEDIUM | 배포 속도 및 안정성 향상 |

## Environment
환경 변수 및 설정 관리

| Impact | Description |
|--------|-------------|
| CRITICAL | 보안 민감 정보 보호 |
| HIGH | 환경별 설정 분리 |
